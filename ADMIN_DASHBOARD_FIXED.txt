================================================================================
                        TROUBLESHOOTING GUIDE
================================================================================

MODAL NOT OPENING
─────────────────────────────────────────────────────────────────────────────
Symptoms:  Click button but nothing happens
Solutions:
  1. Check browser console (F12 → Console tab)
  2. Look for JavaScript errors
  3. Verify onClick={setShowModal(true)} is correct
  4. Check that Modal component is imported
  5. Try refreshing the page (Ctrl+Shift+R)

DATA NOT SAVING
─────────────────────────────────────────────────────────────────────────────
Symptoms:  Form submits but data doesn't appear in table
Solutions:
  1. Check network tab (F12 → Network)
  2. Look for API response (should be 200 OK)
  3. Verify backend is running on port 3001
  4. Check NEXT_PUBLIC_API_URL in .env
  5. Check browser console for error messages
  6. Restart backend server

IMAGE NOT SHOWING PREVIEW
─────────────────────────────────────────────────────────────────────────────
Symptoms:  Select image but preview doesn't show
Solutions:
  1. Verify image file is selected
  2. Check browser console for errors
  3. Try different image file
  4. Check file size (max should be reasonable)
  5. Verify browser allows FileReader API

CATEGORIES NOT SHOWING IN DROPDOWN
─────────────────────────────────────────────────────────────────────────────
Symptoms:  Dropdown is empty or shows "No categories"
Solutions:
  1. Check Categories page - add at least one category first
  2. Refresh menu page after adding category
  3. Check browser console for errors
  4. Verify API is returning categories (check Network tab)
  5. Check that fetchData() runs on page load
