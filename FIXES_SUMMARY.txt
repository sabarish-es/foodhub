❌ TROUBLESHOOTING - TOP 5 ISSUES
================================================================================

1. "Invalid credentials" Error
   ├─ SOLUTION: npm run db:create-admin
   ├─ VERIFY: mysql -u root -p -e "SELECT * FROM users;"
   └─ DETAILS: See TROUBLESHOOTING.md

2. Database Connection Failed
   ├─ SOLUTION: npm run db:init
   ├─ VERIFY: mysql -u root -p restaurant_management
   └─ DETAILS: See TROUBLESHOOTING.md

3. Port 3001 Already in Use
   ├─ SOLUTION: Change PORT in backend/.env
   ├─ VERIFY: lsof -i :3001
   └─ DETAILS: See TROUBLESHOOTING.md

4. Module Not Found (mysql2)
   ├─ SOLUTION: npm install
   ├─ VERIFY: npm list mysql2
   └─ DETAILS: See TROUBLESHOOTING.md

5. CORS or API Errors
   ├─ SOLUTION: Check NEXT_PUBLIC_API_URL in .env
   ├─ VERIFY: curl http://localhost:3001/api/health
   └─ DETAILS: See TROUBLESHOOTING.md
